# Changelog

이 프로젝트의 모든 주요 변경 사항은 이 파일에 기록됩니다.

## [1.0.4] - 2026-05-09
### Changed
- GitHub Actions 워크플로우 임시 제거: AI Studio 권한 이슈(Workflows permission)로 인해 깃허브 푸시가 블락되는 문제를 해결하기 위해 해당 파일을 제거함.
- 프로젝트 최신 상태가 정상적으로 동기화되도록 조치.
- 권한 재설정 또는 수동 추가 가이드를 README에 기록.

## [1.0.3] - 2026-05-09
### Fixed
- GitHub Pages 배포 시 빈 화면(Blank Page) 문제 해결 (`base: './'` 설정).
- `gh-pages` 패키지 추가 및 배포 스크립트(`npm run deploy`) 연동.
- `.nojekyll` 파일 추가로 자산 로딩 최적화.

## [1.0.2] - 2026-05-09
### Added
- 퍼블리싱 준비 완료: 외부 환경 변수(`GEMINI_API_KEY`) 기반의 AI 호출 로직 최적화.
- API 보안 가이드: `Gemini API Key_0509` 신규 키 적용 지침 추가.

## [1.0.1] - 2026-05-09
### Changed
- 브랜드 명칭 변경: "소셜앤비즈" -> "AIARA Space".
- 디자인 테마 변경: 블루 계열에서 앰버/오렌지(이미지 기반) 계열로 업데이트.

## [1.0.0] - 2026-05-09
### Added
- 초기 애플리케이션 개발 완료.
- 10개 카테고리, 100개의 비즈니스 프롬프트 데이터 구축 완료 (`constants.ts`).
- Gemini 3 Flash Preview 모델을 이용한 프롬프트 최적화 기능 (`geminiService.ts`).
- 다크 테마 기반의 반응형 사이드바 UI 적용.
- 한국어/영어 다국어 전환 시스템.
- 클립보드 복사 기능 및 애니메이션 피드백.
- `README.md`, `PRD.md`, `CHANGELOG.md` 문서화 완료.

### Fixed
- `@google/genai` 라이브러리의 최신 인터페이스(`GoogleGenAI`)에 맞게 서비스 로직 수정.
- 프롬프트 데이터 개수가 18개에서 100개로 정상 업데이트됨.
